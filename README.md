# AlemBD – German School Database
## 🏷 Project Overview

AlemBD is a web database created to manage the daily operations of a fictional German language school. It allows staff to easily handle student information. 
This project was part of a high-school thesis-like assignment, designed to be intuitive for users without advanced technical knowledge.

## ⚡ Features

- Add, update, and delete student records
- Manage courses and class schedules
- Track student enrollment and progress
- Simple interface for staff to perform daily operations

## 🛠 Technologies Used

PHP – backend scripting
HTML – front-end structure
SQLite – database engine
SQL – queries and data manipulation

Note: This project reflects early database design learning. Some structures may not fully follow advanced relational modeling practices, 
but it provides a functional and easy-to-use system for daily management tasks.
